The Ceremonial Whetstone
This is a rather mysterious object whose purpose is uncertain. It consists of a whetstone and, probably fixed to one end of this, a bronze stag and ring. The whetstone, which is a stone used to sharpen cutting tools, had not been used. Some experts have decided that this object might have been a sceptre, which is an object associated with kingship that symbolizes royal authority. It may well have been a symbol of power and part of the royal regalia.
